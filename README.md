# Number Analyzer

A small web page that analyzes a number input and displays:
- Whether it's even or odd
- Whether it's positive or negative
- Its square
- Whether it's prime

## Files
- `index.html` — minimal UI (input, buttons, output area)
- `script.js` — main logic and event handlers
- `style.css` — simple styles

## Usage
1. Open `index.html` in your browser (double-click or serve with a static server).
2. Enter a number in the input field.
3. Click the **Check** button or press the `Enter` key to analyze the number.
4. Click **Clear** to reset the input.

## Notes
- The input accepts negative numbers. Pressing Enter triggers the same check as the **Check** button.
- If you don't see negative numbers or output, ensure JavaScript is enabled in your browser and that `script.js` is loaded.
